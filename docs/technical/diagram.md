**2024年2月末時点**

## 配線図の見方

XLR接続(マイク用接続)の線は**太線**、シールド接続の線は**破線**で示しています。
また、先端が●の端子は現在使用中、×の端子は未使用です。

**例**

```mermaid
graph TB
    マイク===ミキサー
    IF-.-スピーカー
```

## スタジオ

```mermaid
graph RL
    subgraph 調整室
        CDデッキ---ミキサー
        レコーダー出力---ミキサー
        ミキサー---スタジオ用アンプ
        ミキサー---レコーダー入力
        ミキサー---放送設備
        ミキサー===モニタースピーカー
        ミキサー<-.->オーディオインターフェイス
        オーディオインターフェイス---iPad
        ミキサー==oマイク7
        スタジオ用アンプ===プッシュトークマイク
    end
    subgraph スタジオ
        ミキサー==oマイク1
        ミキサー==xマイク2
        ミキサー==xマイク3
        ミキサー==xマイク4
        マイク5===ミキサー
        マイク6===ミキサー
        スタジオ用アンプ---スタジオ用スピーカー
    end
```
